# Notification System Frontend

## Overview
This is a React-based frontend application for a Notification System.

## Features
- Dashboard view
- Notifications page
- Routing using React Router

## Tech Stack
- React
- Vite
- JavaScript

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`

## Folder Structure
- src/pages → Dashboard & Notifications
- src/services → API calls
- logging_middleware → Logging logic

## Note
This project follows proper folder structure and coding standards.
